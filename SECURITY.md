# Security Policy

## Reporting a Vulnerability

- Please email Toby Schmeling at nagual69@gmail.com with a description of the issue, steps to reproduce, and any relevant logs.
- Do **not** open public GitHub issues for suspected vulnerabilities.
- You can expect an acknowledgment within 3 business days.

## Supported Versions

Security fixes will be applied to the latest release. Critical fixes may be backported to previous minor versions if feasible.
